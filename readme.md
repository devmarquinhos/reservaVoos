## :computer: Projeto

Sistema que retorna ao usuário todas as informações dos voos disponíveis daquele dia, caso o usuário seja um administrador do sistema, o mesmo pode editar, apagar ou registar novos voos no sistema.

## :rocket: Tecnologias

### Para o Front-End
- HTML e CSS
- JavaScript
- Figma

### Para o Back-End
- Python
- SQLite
- SQLAlchemy
- Flask
- Git e Github
